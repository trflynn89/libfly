Detailed Description

template<typename StringType, typename... ParameterTypes>
class fly::detail::BasicFormatParameters< StringType, ParameterTypes >

A container to hold references to variadic format parameters without copying any of them.

template<typename T , size_t N>
std::optional< T > fly::detail::BasicFormatParameters< StringType, ParameterTypes >::get ( std::size_t  index) const

Visitor to provide runtime access to the stored parameter at the provided index. If the index is found, and is convertible to the desired type, returns a copy of the found format parameter.

This is only allowed for integral format parameters. Attempting to copy other format types is forbidden.
