Detailed Description

template<typename T, T Base, char... Literals>
struct fly::detail::ParserBase< T, Base, Literals >

Structures to parse a sequence of characters and convert the result to an integer literal of a desired type.

Parsing occurs in two phases. First, any base-specifying prefix is parsed. By default, base-10 is assumed, and specializations are used for other bases. Next, the remaining characters are parsed base on the determined base and converted to the desired type.

Template Parameters
TThe desired integer literal type.
BaseThe base of the integer literal being parsed.
LiteralsVariadic list of characters to parse.
