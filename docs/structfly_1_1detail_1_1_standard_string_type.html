Detailed Description

template<typename T>
struct fly::detail::StandardStringType< T >

Concept that is satisfied when the given type is like a supported std::basic_string specialization. A type is "like" a std::basic_string specialization if it is that specialization itself, a C-string equalivent, or a std::basic_string_view specialization.
